# CSV TO JSON

- step 0. got to https://nodejs.org/en/ and download NODE JS
- step 1. using your command line cd into the folder
- step 2. you need to download the modules. In your command line type "npm install" hit enter
- step 3. move your .csv file into the csv-to-json folder
- step 4. rename your .csv to file to "csv.csv"
- step 5. in your command line type "node index.js" hit enter

## FINISHED
you have a new file json
